# **App Name**: DevFolio

## Core Features:

- Dynamic Content Display: Render key sections like Hero, About Me, Skills, and Working Approach with static content provided in a professional, clean manner.
- Interactive Project Showcase: Display developer's featured projects in a modern card/grid layout with individual details, including tech stack and role.
- Direct Communication Links: Provide easily accessible and clickable links for email, GitHub, and LinkedIn to facilitate direct contact.
- Responsive Design: Ensure the entire portfolio website is fully responsive, offering an optimal viewing and interaction experience across all device sizes (desktop, tablet, mobile).
- Subtle UI Animations: Implement light hover effects for interactive elements and smooth transition animations to enhance user engagement without being distracting.
- AI-Powered Portfolio Assistant: Integrate a generative AI tool to answer visitor questions about the developer's projects, skills, or working methodology based on the portfolio content, demonstrating their use of AI for analysis and problem-solving.

## Style Guidelines:

- Color scheme: Dark theme to convey professionalism and modernity. The primary accent color will be a deep, vibrant blue (#4D89ED), reminiscent of technology and reliability. The background will be a dark grayish-blue (#171B1E) providing a sophisticated canvas. A secondary accent will be a brighter, analogous sky-blue (#74D5ED) used for subtle highlights and interactive elements, ensuring strong contrast and visual appeal against the dark background.
- Headline and body font: 'Inter', a clean and objective grotesque sans-serif. Its modern and neutral look provides excellent readability and aligns perfectly with a professional, clean aesthetic.
- Use a set of modern, minimalist, line-based icons throughout the site. These will maintain a clean aesthetic and ensure clarity, particularly for social links and section indicators.
- The layout will be clean, structured, and uncluttered, focusing on a minimalist approach. Projects will be presented using an interactive card/grid UI for clear organization and visual hierarchy. Emphasis on spacious design and intuitive flow, with full responsiveness for all screen sizes.
- Implement subtle animations and transition effects, such as soft hovers on interactive elements (buttons, project cards) and smooth page transitions. These will enhance user experience by providing visual feedback without overwhelming the user or detracting from content.
